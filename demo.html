About Bar Length Finding

The bar length finder uses a binary search algorithm to determine the optimal length for a uniform (uncut) bar to produce the target fundamental frequency. The relationship between bar length and frequency is inverse: longer bars produce lower frequencies.

Key points:

  • Uses 2D Timoshenko beam FEM for fast computation
  • Typically converges within 10-20 iterations
  • Achieves accuracy within ¬±1 cent
  • Starting point for multi-modal optimization

Note: Optimization is computationally intensive. For demonstration purposes, this uses a small population and few generations. Production work would use larger parameters.

Step 3: 3D FEM Verification

Finally, we verify the optimized design using 3D solid element FEM for maximum accuracy.

3D FEM Analysis Features:

  • 8-node hexahedral (brick) solid elements
  • Full 3D stress and strain distribution
  • Adaptive mesh refinement near undercuts
  • Mode classification (bending vs. torsional)
  • Typically within 1-2% of measured frequencies

Use compute_frequencies_3d() function for production verification of optimized designs.

Complete Workflow Summary

  1. Bar Length Finding: Determine initial bar length for target fundamental frequency
  2. Evolutionary Optimization: Find optimal undercut positions and depths for multi-modal tuning
  3. 3D FEM Verification: Validate final design with high-accuracy 3D analysis
  4. Physical Prototyping: Export geometry for CAD/CAM and create actual bar
  5. Fine-tuning: Measure frequencies and make minor adjustments as needed

About This Library

This is a JavaScript port of the multi-modal bar tuning optimization library, originally implemented in Python. It provides tools for designing and optimizing percussion instrument bars (marimbas, xylophones, vibraphones) to achieve precise harmonic tuning.

Features:

  • Comprehensive materials database with metals and tonewoods
  • Standard tuning presets for various instruments
  • Note-frequency conversion utilities
  • Frequency analysis and error calculation
  • FEM-based frequency computation (3D solid elements)
